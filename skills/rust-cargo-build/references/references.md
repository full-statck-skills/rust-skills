# Cargo Build References

## Key commands
| Command | Purpose |
|---------|---------|
| cargo build | Build project |
| cargo check | Check (no bin) |
| cargo test | Run tests |
| cargo doc | Generate docs |
| cargo publish | Publish crate |
| cargo add | Add dependency |
| cargo tree | Show dep tree |
