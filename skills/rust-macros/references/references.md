# Macro References

## Key concepts
| Concept | Description |
|---------|-------------|
| `macro_rules!` | Declarative macros |
| `#[proc_macro_derive]` | Derive macros |
| `#[proc_macro_attribute]` | Attribute macros |
| `#[proc_macro]` | Function-like macros |
| `syn` crate | Parse Rust syntax trees |
| `quote` crate | Generate TokenStreams |
