# Project Structure References

## Key concepts
- Package: one Cargo.toml, one or more crates
- Crate: binary (main.rs) or library (lib.rs)
- Module: file-based, controlled by `mod`/`pub`/`use`
- Workspace: shared Cargo.lock across packages
