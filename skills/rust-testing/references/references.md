# Testing References

## Key macros
| Macro | Purpose |
|-------|---------|
| `#[test]` | Mark test function |
| `assert!(expr)` | Assert true |
| `assert_eq!(a, b)` | Assert equal |
| `#[should_panic]` | Expect panic |
| `#[ignore]` | Skip test |
